Annuaire EQL Documentation

Bienvenue sur la documentation de l’annuaire EQL. Nous vous remercions pour l’utilisation de notre logiciel.Nous allons vous guider pour vous aider à le prendre en main en vous exposant l’ensemble des fonctionnalités qui s'offrent à vous. Ce guide est mis à jour continuellement et est disponible en ligne. Vous avez également la possibilité de contacter l’assistance auprès du service informatique de votre entreprise grâce à ce lien.

Consultation

Lors de l’ouverture du logiciel dans la partie centrale vous pouvez retrouver le tableau listant l’ensemble des stagiaires ayant étudié à l’eql. De haut en bas chaque ligne correspond à un stagiaire. De droite à gauche vous retrouvez le nom , le prénom, le département, la promo, et l’année. Le tri des différents stagiaires est par défaut effectué sur le nom par ordre alphabetique.Si vous souhaitez modifier cet ordre vous pouvez cliquer sur l’entête de la colonne. Exemple: si vous cliquer sur l’entête de la colonne nom le trie du tableau s’effectuera dans l’ordre aphabetique inversé. Vous pouvez sélectionner chaque stagiaire en cliquant ce qui mettra davantage en évidence la ligne sélectionnée.

Recherche

Dans le panneau de droite vous retrouverez l’espace dédié à la recherche d’une liste de stagiaires selon des critères précis depuis n’importe quelle caractéristique. Pour ce faire vous avez à votre disposition la possibilité de taper le champ avec lequel vous souhaitez le comparer. En dessous vous retrouvez une liste déroulante qui vous permet de sélectionner sur quelle critère vous souhaitez comparer le champ texte rempli au dessus. Le bouton recherche vous permet enfin de lancer la recherche. Dans le panneau central (consultation) vous retrouverez l’ensemble des résultats de votre recherche. Le tri des éléments recherché est une nouvelle fois classé par ordre alphabétique sur le nom. Exemple: Vous souhaitez rechercher la liste des stagiaires ayant étudié en 2010. Vous allez entrer “2010” dans le champ de recherche. Sélectionner l' année dans le menu déroulant et enfin cliquer sur le bouton rechercher. Le résultat de votre recherche s’affiche maintenant dans le panneau central.

Création

Le logiciel vous offre la possibilité de pouvoir en plus de consulter les stagiaires présent dans la base de données de pouvoir également en ajouter de nouveau. Pour cela il vous faut cliquer sur le bouton “ajouter stagiaires” en bas à droite du panneau de droite de l’application. Une nouvelle vue s’ouvre sur la gauche de l’application . Cette vue vous demande de remplir tous les champs indispensables pour constituer un nouveau stagiaire. Les informations à renseigner sont les suivantes : {nom, prénom, département, promotion, année d'étude}. Les informations dûment remplies vous cliquerez sur le bouton enregistrer. Un nouveau stagiaire à maintenant été ajouté à la base de données. Il apparaît directement dans le tableau de la partie centrale. Exemple: Vous souhaitez enregistrer un nouvel utilisateur Dupont Michel habitant dans le 92 participant à la formation ai109 de 2021. Vous cliquerez sur le bouton nouveau stagiaire (panneau de droite) Entrez les informations dans cet ordre. nom : “dupont”, prenom: “michel”, departement : “94”, formation: “ai109”, année:”2021” Cliquez enfin sur enregistrer, vérifier l’ajout dans le tableau de la partie centrale.

Export PDF

Il vous est offert la possibilité d’exporter les informations d’un stagiaire ou d’une liste de stagiaire sous la forme de PDF. Vous pouvez à la fois exporter un document contenant l’ensemble des stagiaires ou n’extraire uniquement que le résultat de n’importe quelle recherche simplement en cliquant sur le bouton export PDF en bas à droite dans le panneau de droite (panneau droit). Il vous est offert la possibilité de choisir l’emplacement ou sera enregistrer votre document. Exemple: Vous souhaitez exporter les informations concernant les stagiaires présents en 2010. Vous allez rechercher dans le panneau de gauche (champ de recherche : “2010”, sélectionner: “Année”) cliquer sur recherche. Cliquez enfin sur export PDF sélectionner le dossier dans lequel vous souhaitez l’ajouter puis valider.
